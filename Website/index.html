Customizable

If you know a little bit of CSS you will be able to customize Mogi however you want, changing elements, classes and creating themes. Every component is well documented and OpenSource, and it is going to help you a lot, trust me.

Responsive

Mogi is completely responsive, this means that all the websites made with it can be viewed perfectly from every kind of device, including tablets and smartphones.

Multi theme

On Mogi there are 3 default themes which can be selected by users: light, dark and Mogi. If you want you can create your own following the documentation.
