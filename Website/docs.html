Download the package...

First of all integrate the package, in order to do this, click on the item on the menu "Download" and look for the newest version. Once you've done that, extract the zip file and open index.html, then copy the code from here, and paste it inside of the body.
Mogi has 3 default themes, which you can find on the bottom right of the screen. It uses a Js script to switch between the Css pages, so if you'd like to create your own theme, just create a copy of an existing css page (dark.css, light.css or mogi.css) and edit whatever you want.

Classes

custom element classes

Here's a list of classes that you can add to almost every element

code <!-- Gives paragraph a code looking --> boxed <!-- Puts the content inside of a box --> shadow <!-- Puts shadows behind the elements --> light-text <!-- The text becomes light --> dark-text <!-- The text becomes dark --> stable <!-- Makes a paragraph a little bit bigger --> parallax <!-- Gives a background image a parallax effect --> sticky <!-- The element becomes sticky (used for headers) -->

Create a custom theme

how to create your personal theme

First of all create a file inside of the css folder, called yourname.css
Then copy the code from another theme (like dark.css or light.css) and paste it in your page.
At this point you can start editing the style however you want.
Once you've finished this, go to the bottom of every page (the html file), look for the "script", then add another if statement like this:

if(document.getElementById("yourtheme").selected) { swapStyleSheet('css/yourtheme.css') }

Then go to the top of the page, you look for the fixedbutton id, and then add an option with your theme name

<option value="themeName" id="themeName">themeName</option>
