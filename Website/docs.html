Integrate the style

Integrate this style in order to use the components. If you've already downloaded the complete folder, you don't have to put this code inside the index as it is already present.

<head> <meta charset="UTF-8"> <title>Mogi Framework - Fast and Free</title> <link rel="stylesheet" href="css/custom.css"> <link rel="stylesheet" href="css/style.css"> <meta name="viewport" content="width=device-width, initial-scale=1"> <link href="https://fonts.googleapis.com/css?family=Lato&display=swap" rel="stylesheet"> <link rel="stylesheet" href="https://use.fontawesome.com/releases/v5.9.0/css/all.css" integrity="sha384-i1LQnF23gykqWXg6jxC2ZbCbUMxyw5gLZY6UiUS98LYV5unm8GWmfkIS6jqJfb4E" crossorigin="anonymous"> </head>
